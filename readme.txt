How to use:

1. Download BetterDiscord. Bandaged BD works for this theme. 
2. Go into settings and under "Bandaged BD" find "Themes"\
3. Once your here, click "Open theme folder" 
4. Drag the CSS file into the folder that you unzipped from the .ZIP file you can download on my GitHub. 
5. Enjoy!
